# Branded Components

This folder contains brand-customized components exported from Figma.

## For Designers:
- Export components from Figma as React/TypeScript code
- Place exported files here
- Push to the `design` branch for review

## For Developers:
- These are READ-ONLY components
- Import from this folder in your projects
- Do NOT modify these files

## Workflow:
Designer exports → Push to design branch → Review → Merge to master → Developers pull
